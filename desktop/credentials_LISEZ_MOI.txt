Pour activer Gmail, Google Docs, Sheets et Calendar, vous avez
besoin d'un fichier credentials.json depuis Google Cloud Console.

── Étapes ─────────────────────────────────────────────────────

1. Allez sur : https://console.cloud.google.com/

2. Créez un projet (ou sélectionnez-en un existant)

3. Activez ces APIs dans "Bibliothèque d'APIs" :
   - Gmail API
   - Google Docs API
   - Google Drive API
   - Google Sheets API
   - Google Calendar API

4. Dans "Identifiants" → "Créer des identifiants"
   → Choisissez "ID client OAuth 2.0"
   → Type d'application : Application de bureau (Desktop app)
   → Nom : JARVIS Morphoz

5. Téléchargez le fichier JSON
   → Renommez-le "credentials.json"
   → Placez-le dans ce dossier : desktop/

6. Lancez JARVIS et dites "lis mes emails" ou "crée un document"
   → Une fenêtre de connexion Google s'ouvrira
   → Connectez-vous avec votre compte Google
   → Un fichier token.pickle sera créé automatiquement

── Notes ──────────────────────────────────────────────────────

- Le fichier token.pickle stocke votre session (pas besoin de
  vous reconnecter à chaque démarrage)

- Si vous changez de compte Google, supprimez token.pickle

- Votre credentials.json est PERSONNEL — ne le partagez pas
